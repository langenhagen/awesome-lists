How to Build the Project:

Write or maintain the markdown files. Then, build your HTML files manually via ./convert-markdown-to-html.sh.

Best view the files on a server in order to allow for the session-storage to set the bright/night theme throughout your pages. Locally, you can run e.g.: 

python -m http.server
