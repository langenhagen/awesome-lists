Awesome Tools

A list of neat apps that are worth checking out.

CLI

  • rlwrap: A simple repl-with-history wrapper for any given CLI app https://github.com/hanslub42/rlwrap
  • reposet: A toolset to manage many git repos, written by me https://github.com/langenhagen/reposet
  • xdotool: Simulates keyboard- and mouse input and more for X11 window control; also as a C++ library https://github.com/jordansissel/xdotool

TUI

  • tig: A small, hackable git client written in C++ https://github.com/jonas/tig

GUI

  • dmenu: A small, performant, hackable dynamic menu X11 written in C++ https://tools.suckless.org/dmenu/
